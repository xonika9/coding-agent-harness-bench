Реализуй функцию `run` (стековая виртуальная машина) в `src/vm.ts` строго по `spec.md`.

Правила:
- Пиши реализацию в `src/`. Файлы в `test/` менять НЕЛЬЗЯ.
- Тесты: `node --test 'test/**/*.test.ts'` — должны стать зелёными.
- Типы: `tsc --noEmit -p tsconfig.json` — чисто.
- Node исполняет `.ts` в режиме strip-only: без `enum`, parameter properties и `namespace`.
- Решение проверяется в том числе на скрытых крайних случаях из `spec.md`, которых нет
  в видимых тестах, поэтому следуй спецификации буквально, а не только видимым примерам.
- Работай самостоятельно до результата, ничего не спрашивая.
