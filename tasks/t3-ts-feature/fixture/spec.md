# Спецификация: parseDuration

`export function parseDuration(input: string): number` — возвращает суммарную
длительность в миллисекундах.

## Единицы (множитель к мс)
- `ms` = 1
- `s`  = 1000
- `m`  = 60000
- `h`  = 3600000
- `d`  = 86400000

## Формат
- Вход — один или несколько токенов вида `<целое><единица>`, записанных подряд:
  `"1h30m"`, `"500ms"`, `"2d"`, `"1h30m15s"`.
- Целые неотрицательные (`0` допустим). Без знаков и без дробей.
- Токены суммируются слева направо.
- Ведущие и хвостовые пробелы игнорируются. Пробелов внутри строки нет.
- Любой вход, не соответствующий этой грамматике (пустая строка, неизвестная
  единица, число без единицы, дробь, посторонние символы), должен бросать `Error`.
