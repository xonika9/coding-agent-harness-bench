В этом Python-проекте часть тестов падает из-за одного бага в коде.

Задача: найди и почини баг, чтобы вся тестовая сюита проходила зелёной.

Правила:
- Правь только код в `src/`. Файлы в `test/` менять НЕЛЬЗЯ.
- Запуск тестов: `python3 -m unittest discover -s test -t . -p 'test_*.py'`
- Готово, когда все тесты проходят.
